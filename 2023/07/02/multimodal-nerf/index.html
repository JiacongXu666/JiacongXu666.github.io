Multimodal-NeRF-Reading

1. Multimodal Neural Radiance Field (Amazon)

Paper

Method

Point Cloud Extraction

The relative transformation matrix can be estimated for the sets of 2D images (rgb, infrared). Project the points on each image to the 3D space by the matrix and depth (estimated by Depth Estimation Network), then we will have the point cloud for each modality.

Point Cloud Registration

Extract the features for both of the two point clouds by a feature extraction network and align two spaces by RANSAC.

Loss

Following PointNeRF.

Dataset

In addition to the RGB images, M2DGR also provides thermal-infrared images, which capture the 2D geometry information in greyscale images with infrared sensors. 165 infrared images along with 280 RGB images are selected.

Performance

More modalities will lead to better performance (compared with naive NeRF and PointNeRF).

2. Cross-Spectral Neural Radiance Fields (3DV 2022)

Paper

The Objective is to unify the geometry (volume density) between different modalities and this papaer focuses on the learning of a unified representation not the improvement of synthesis performance. The issue is that they did not show the advantages of usage of other spectral images (such as some severe situations).

Method

X-Nerf

Assume that the volume density is shared between different modalities (rational for infrared but not good for x-ray), so the network for $\sigma$ is fixed but the networks for color are not share their weights.

Pose Estimation

When collecting the images, 3 cameras are mounted on a rigid frame, so relative poses between them are fixed. Exploit COLMAP only on a single modality, and treat the relative pose as learnable parameters.

NXDC: Normalized Cross Devices Coordinate

Constrain ratios fx/W and fy/H to be fixed across devices in order to make use of NDC for all the devices with different paramters.

Loss

Simple total squared error between the rendered and true pixel values.

Dataset


Self-collected, but there is no images for severe situations.

Performance
